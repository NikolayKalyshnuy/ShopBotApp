Возможности Telegram-бота:

    1. Пользователь выбирает товар, количество и кладет в корзину. Товары настраиваются в админ панели.
    2. Пользователь должен заполнить форму с заданными полями в панели администратора.
3. Пользователь выбирает пункты самовывоза для заказа из списка, заданного в админке.
4. Пользователь получает push-уведомление из пункта выдачи с обновлениями. Push отправляется через панель администратора.
5. Пользователь получает кнопку со ссылкой на направление к Google Maps, если получено push-уведомление о заказе.

6. Оплата наличными для клиентов. В будущем будет добавлена полоса, и только специальные разблокированные клиенты смогут платить наличными при получении. Все остальные должны использовать Stripe


Панель администратора

    1. Роли пользователей
2. Проверить новые, ожидающие, выполненные заказы.
3. Отфильтруйте заказы по лицам, товарам, пунктам выдачи.
    4. Список пользователей и отображение сведений о пользователе.
    5. Отмена, редактирование, удаление, добавление новых заказов.
6. Покажите детали заказа с личными данными и оплатой, если она сделана или нет.
7. Группируйте пользователей по тегам и пользовательским столбцам БД
8. Отправляйте сообщения Telegram в группы.
